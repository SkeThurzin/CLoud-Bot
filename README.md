### 💫 Iniciando:

- `1.` Clone ou instale o projeto.
```bash
git clone https://github.com/SkeThurzin/Cloud-Bot.git
```

- `2.` Abra o terminal manualmente ou `Ctrl+Shift+`

- `3.` No Terminal execute:
```bash
npm install
```
### Or
```bash
npm i 
```

- `4.` Depois de instalar as dependecias execute no terminal:
```bash
node .
```
### Or
```bash
node index.js
```
